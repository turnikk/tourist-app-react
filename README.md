# Intro

A small app that helps a tourist discover new and exciting places around them. The user should be able to query locations based on categories like; restaurants, amusement parks, shopping malls, etc. These categories should be displayed in a dropdown and a search bar should be provided for the user to type in a location.

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Available Scripts

In the project directory, you can run:

### `yarn start`

Runs the app in the development mode.\

The page will reload if you make edits.\
